积分说明

1. 什么是积分？

有调积分是针对有调注册用户的奖励体系。当您通过新浪微博或微信账号登录有调后，即可开启您的有调积分之旅。

2. 如何获得积分？

登录App、每日签到、分享内容及邀请好友等操作都可以获取积分。获得的积分可以在积分商城中兑换优惠券、礼品卡以及有调精选好物等礼品。具体的积分规则及获得方式如下：

每日签到：

  • 连续签到第1天-第5天：每日签到积分+3；
  • 连续签到5天：额外获得15积分；
  • 连续签到第6天-第30天：每日签到积分+5；
  • 连续签到10天：额外获得20积分奖励；
  • 连续签到20天：额外获得50积分奖励；
  • 连续签到30天：额外获得100积分奖励；
  • 连续签到第31天及以上：每日签到积分+10；
  • 连续签到31天后每连续签到15天：额外100积分奖励；
  • 当连续签到中断后，则回到初始签到状态重新累计连续签到天数；
  • 签到系统的时间计算，以北京时间为准。

邀请好友下载有调：

  • 每邀请一名好友下载有调并成功输入你的邀请码，你与好友均可获得积分+25；
  • 当用户累计邀请5个好友的时候，可额外获得50积分的奖励；当累计邀请10个好友的时候，可再获得100积分的奖励；当邀请好友数超过10之后，每5个好友额外奖励50积分。
  • 邀请的好友需为真实有效用户，邀请行为不允许作弊，一经发现，会做拉黑处理。
  • 最终解释权归有调APP所有。

分享：

  • 每天成功将有调内容分享到其他平台：每次获得积分+3，最多3次。

其他操作：

  • 新用户首次登录成功：赠送30积分；
  • 首次开启每日签到提醒：赠送20积分；
  • 首次在积分商城选择“心动好礼”，设为积分目标：赠送20积分；
  • 首次在积分商城兑换优惠码/礼品：赠送20积分；

3. 心动好礼是什么？我要如何设置？是否可以更改？

心动好礼是从有调积分商城的积分好礼中，自行选择并设定的积分兑换目标。当您选择某一积分好礼后，可点击最下方“设为心动好礼”按钮，确认后该好礼将成为您的目标礼物并出现在“我的”页面的进度条中。进度条的红色圆环表示当前积分进度，当红色圆环头尾相遇时，即可成功兑换心动好礼啦！

如想修改当前心动好礼，在“我的”页面点击您的心动好礼图片即可修改；也可在积分商城中选择心仪的积分好礼，点击好礼详情页最下方“设为心动好礼”按钮进行替换。

所有积分好礼都是从有调数千款好物中选出的最具人气的商品，只要用户的积分达到兑换标准，即可成功兑换该好礼！

4. 积分商城内的优惠券怎么兑换？如何领取？

用户可在积分商城的优惠券频道，选择您所需要的商家的优惠券，点击最下方“立即兑换”按钮，确认兑换操作后系统将扣除相应的积分，即可获得优惠券码。

长按优惠券码复制成功后，选择“去商家兑换”则可进入相应页面。

您也可在我的页面->积分商城->兑换记录中，查看到以往的兑换历史。如果兑换后没有立即使用，在兑换记录中点击相应的优惠券也可查看优惠券码。

为了保证有调用户的权益，优惠券对领取券码的IP、ID做了相应限制。如您处在公共区域上网，您所属的IP领取过优惠券后，对于限制领取次数的优惠券您将无法再领取。

5. 我成功兑换了积分商城的好礼，如何领取？是否可以更改？

当您成功兑换了某一积分好礼时，将会提供有调官方店铺链接供您兑换。

您只需跳转到有调官方店铺内拍下该链接中的商品、填写收货地址及联系方式、并在留言处注明您的有调帐号昵称+兑换礼品名称即可。

我们的工作人员会在核实无误后尽快将实物礼品寄送给您。

积分好礼及优惠券一经兑换后，均不可更改，所以各位调友在兑换前要再次确认哟~~

6. 其他问题

有调积分在年底不会清零，请各位调友放心使用。

如有调用户在积分积累过程、或商品兑换过程中遇到上述无法解答的情况，请联系有调小秘书进行咨询。

有调小秘书位于“我的”页面内、积分进度下方，我们将及时处理。小秘书的在线时间为每天8点至22点。
